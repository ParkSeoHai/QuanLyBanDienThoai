# QuanLyBanDienThoai
Đề tài: Quản lý bán hàng điện thoại/ Môn học: Lập trình hướng đối tượng(OOP)/ Trường Đại Học Công Nghệ Đông Á/ Năm học 2022 - 2023/ Sử dụng Java swing.

- Database: Oracle.
- Chức năng:
  - Đăng nhập
  - Quản lý nhà cung cấp
  - Quản lý sản phẩm
  - Quản lý nhập xuất hàng hóa
  - Thống kê báo cáo
